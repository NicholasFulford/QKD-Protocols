# QKD Protocols — BB84 and E91

This repository contains implementations of the BB84 quantum key distribution protocol using Qiskit, investigating eavesdropping models, noise analysis, and security. Further work will extend this to the E91 protocol.

## Current status
- [ ] BB84 protocol with intercept-resend eavesdropper analysis <-- *in progress*
- [ ] Noise and channel analysis
- [ ] Security and key rate analysis
- [ ] E91 protocol with CHSH inequality test
