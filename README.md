# QKD Protocols — BB84 and E91

This repository contains implementations of the BB84 and E91 quantum key distribution protocols
using Qiskit. Further work will include eavesdropping models, noise analysis, and security discussion.

## Current status
- [ ] BB84 protocol with intercept-resend eavesdropper analysis <-- *in progress*
- [ ] E91 protocol with CHSH inequality test
- [ ] Noise and channel analysis
- [ ] Security and key rate analysis
